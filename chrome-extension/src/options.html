AiDD Toolset Options

Enable File Selection

When enabled, you will be able to select specific files from your project to include in the AiDD prompt.
When disabled, the prompt will be generated without file content.

Auto submit prompt when launching SkyDeck.ai

When enabled, the prompt is automatically submitted after pasting into SkyDeck.ai.
When disabled, the prompt is pasted but not submitted, allowing you to review and select a model before sending.

Allow code changes to be applied directly within SkyDeck.ai

When enabled, you will be able to apply code changes from AI responses directly within SkyDeck.ai.

Auto fix Git Diff generated by AI when it is incorrect

When enabled, the extension will automatically attempt to fix incorrect Git diffs generated by AI.
When disabled, you will have the option to manually fix incorrect Git diffs.

Hide content of files added via the File Insertion tool

When enabled, the content of files added via the File Insertion tool will not be displayed (only applied to SkyDeck GenStudio).

Shortcuts

Quickly Open File Selection (Don't Ask Me To Select a Project Folder) This shortcut remembers your previously selected project folder, so it won't ask you to select again.
Use this shortcut to quickly open a file selection dialog and apply a pre-configured template to generate your desired AiDD prompt.
Trigger File Insertion Use this shortcut to trigger the File Insertion Modal, allowing you to insert specific files or code snippets into your current context.
